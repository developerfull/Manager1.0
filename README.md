# Manager Project

O objetivo do projeto é fazer o gerenciamento dos projetos ao qual o meio está envolvido.

## Getting Started

Essas instruções farão com que você tenha uma cópia do projeto em execução na sua máquina local para fins de desenvolvimento e teste. Veja a implantação de notas sobre como implantar o projeto em um sistema ativo.
### Prerequisites

JAVA 
XAMPP|DOCKER
ECLIPSE


### Installing

Baixe o projeto em sua máquina. Cadastre um banco de dados mais o usuário e senha que seja referente as configurações que você implentou nas variáveis de ambiente de seu projeto.
